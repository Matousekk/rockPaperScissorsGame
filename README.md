Simple rock paper scissors game with UI elements, first project.
Computer choice is based on Math.random
Score goes only to 2 or -2
Once player gets to 5 wins, he gets a +1 score, if 10 wins, +2 score, but every loss takes away one win and gives it to computer.
Match consists of 5 rounds, after 5 rounds, match is reset and going from 0 to 5 again.
